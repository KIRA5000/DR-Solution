# cpp-rct
Hyper-V 2016 RCT backup and recovery library

### Build
 - Download and install latest cmake
 - Fork the main repo
 - Clone the forked repo under a folder say c:\cpp-rct
 - From inside a VS 2019 x64 command prompt:

```sh
cd c:\cpp-rct
md build
cd build
cmake ..
cmake --build . --config Release
```
